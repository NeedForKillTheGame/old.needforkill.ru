CTF Online 3
После долгого инактива нашего коммунити, было решено возродить CTF online. Данный ивент будет проходить в субботу 4 октября в 11:00 утра по мск. Регистрация начнется в 10:00 того же дня на нашем irc-канале #nfk@quakenet.org.

Маплист: ctf1

Матчи будут проводится сначала на сервере одной команды, затем - другой. Если победитель не был выявлен по двум встречам, играется третий матч на дедике.

Настройки сервера:
forcerespawn 10
capturelimit 5
fraglimit 0
timelimit 0
sv_maxplayers 4
sv_maxspectators 0
sv_allowjoinmatch 1
sv_allowspectators 0
sv_allowvote 1
sv_allowvote_restart 1
sv_allowvote_fraglimit 0
sv_allowvote_timelimit 0
sv_allowvote_ready 1
sv_allowvote_map 1
sv_allowvote_warmup 0
sv_allowvote_warmuparmor 0
sv_allowvote_forcerespawn 0
sv_allowvote_sync 0
sv_allowvote_sv_teamdamage 0
sv_allowvote_net_predict 0
sv_allowvote_sv_maxplayers 0
sv_vote_percent 100
sv_teamdamage 0
sv_overtime 2
sv_powerup 1
sync 3
warmup 180
warmuparmor 50

+ Запись демок сервером обязательна
