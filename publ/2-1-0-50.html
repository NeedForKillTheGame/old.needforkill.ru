Запуск демки двойным кликом

Статья будет полезна тем у кого много демок, т.к. чтобы посмотреть ту демку которую вы хотите нужно долго ее искать через меню в нфк.
Чтобы запустить демку двойным кликом из любой директории делаем вот что:
 

1. Заходим в любую папку, Сервис - Свойства папки... - Типы файлов


2. Нажимаем кнопку Создать

3. Расширение: ndm, затем ОК


4. Выбираем в списке файлов, только что созданный и нажимаем Дополнительно

5. Далее нажимаем кнопку Создать


6. Появится окно созданиия нового действия



Действие: Play demo (или любое другое)
Приложение, исполняющее действие - выбираем файл nfk.exe в директории нфк
И дописываем к этому +demo "%1"
Вот примерно что должно получится "D:\Games\Need For Kill\NFK.exe" +demo "%1"
Можно также добавить другие комманды например  +gowindow +demo "%1"

6. Нажимаем ОК

Все, теперь можно просматривать демки двойным кликом мыши
