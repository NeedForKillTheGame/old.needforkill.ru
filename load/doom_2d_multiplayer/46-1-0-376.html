Doom 2D Multiplayer 0.6(130)
2009-12-31, 12:06

Doom 2D Multiplayer - это сетевой мультиплеерный римейк старой и известной игры Doom 2D, сделанный на Game Maker. Этот проект создавался для того, чтобы многочисленные фанаты старого доброго Doom 2D смогли вместе поиграть в любимую игру, несмотря на разделяющие их расстояния. По сути игра очень похожа на небезызвестный Doom 2D Arena, только с гораздо меньшим количеством ошибок и недоработок.

Особенности версии 0.6:

  • все снова переписано с нуля;
  • более стабильный сетевой код;
  • всё оружие и почти все предметы из оригинального Doom 2D;
  • навороченный редактор карт и подробная документация к нему;
  • сменные скины для игроков и возможность их создания;
  • ещё какие-то различия, которые при желании и наличии времени найти сможет каждый. :D
