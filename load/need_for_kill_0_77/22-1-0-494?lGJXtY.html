Need For Kill 0.77 Update 42
[ · Скачать удаленно (30 Mb) ] 2010-11-29, 17:15

Игра Need For Kill, последняя версия на данный момент.

Новые возможности:
- Автоматическое обновление NFK.
- Запуск демки двойным кликом из любой папки.
- Запуск редактора карт при двойном клике на файлы .mapa
- NFK сама соединяется с сервером при нажатии на ссылку в браузере nfk://IP (например nfk://178.140.108.197:29991)
- Выбор установленного мода из Launcher'a

Старые моды не работают с NFK 0.77. 
 

Внимание: Если у вас включен UAC и игра установлена в защищенную папку (например Program Files; только для Vista или Windows 7), Launcher.exe необходимо запускать с правами администратора (только когда есть обновления, иначе они не установятся).
