  • 1 GB RAM or more (at least 2 GB for Vista/7/8!)

  • Windows (2000,XP,Vista,7,8), Linux or MacOS with Direct3D/OpenGL compatible drivers

(These are the recommended specifications. The game might run well on slower systems!)
