Обновление дедика (Build 42)


  • Добавлена проверка на версию дедика (старые dll < 42 сборки больше не поддерживаются статистикой)
  • Добавлена проверка на несовместимость режима игры и карты (не полностью)

Добавлены новые команды:


  • stats_enabled - вкл/выкл ведение статистики, по умолчанию = 1
  • stats_autolimits - при включении автоматически делает для tdm timelimit 15, для ctf 0, для dm 10. По умолчанию = 1
  • stats_debugmsg - вывод дополнительной информации в консоль, по умолчанию = 0
Эти настройки необходимо прописывать в файле autoexec.cfg

1) Бывает видишь интересные матчи в статистике, а демок нету. Вручную заливать демку после каждого матча неудобно, да и многие игроки просто не пишут демок.

Такое предложение.. Нельзя ли сделать чтоб дедик автоматически записывал демки, и сохранял их только когда игра завершена (timelimit/fraglimit/capturelimit). Так чтоб можно было посмотреть сыгранный матч в статистике и сразу скачать демо. Старые демки, к примеру, записанные 2 месяца назад - автоматически удалять с сервера, чтоб не засорять, либо ограничение на кол-во демок на сервере (500 демок например, при записи новой, самая старая сразу удаляется).

В пкур моде есть функция автоматической записи демо. Запись начинается автоматически при старте игры, пишется в файл !temp!.ndm. Затем после окончания игры, если игроку нужно, он может просто активировать команду match_save и pqrmod сохранит записанную дему !temp!, но уже с нормальным именем. (Возможно это как-то поможет)

2) Небольшой баг с цветовыми тегами и использованием символа ^.
Вот мой ник в конфиге: ^!<^7S^!^^72^!> ^7Roks
В нфк это выглядит (цвета опущены): <S^2> Roks
В статистике почему-то выглядит: <S72> Roks

^ отсутствует, и вместо него вылазит 7, которая задает цвет цифре 2.

Конечно это не критичный, но всеже баг, с низким приоритетом

Обновление дедика (Build 44)
Новые параметры в vote:


  • !vote stats off [on] - голосование за отключение или включение статистики на сервере (доступно только на Rip #1 All)
  • !vote kick [PlayerID] - кик игрока с сервера

Добавлены новые команды:


  • st_automsg 1/0 - в вармапе выводит сообщение из файла massage.cfg всем игрокам на сервере
  • st_automsg_time 0-999 (по умолчанию 120) - задает периодичность вывода сообщений, в секундах
  • st_allowvote_st_enabled 1/0 - разрешает/запрещает голосование за отключение/включение статистики на сервере
  • Приставка stats_ переименована в st_ (stats_enabled -> st_enabled)


  • Исправлен вылет дедика при входе игрока с пустым ником (например ^1)
