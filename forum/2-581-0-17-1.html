Скачал TRXSV23I.rar, у него утечка памяти жесткая - через некоторое время начинает кушать 100-500 мб и продолжает расти. Пинг при этом тоже нехило скачет, даже если ресурсов на компе достаточно. Настройки добавил в server.cfg из http://needforkill.ru/faq/0-0-8, получилось примерно так:
cpufix 0
// options
log 0
sv_dedicated 1
sv_port 29998
sv_hostname harpyserv
sv_allowspectators 1
sv_maxspectators 4
sv_allow_r_mode 1
sv_mapcycle 1
capturelimit 5
fraglimit 0
timelimit 60
sv_maxplayers 8
sync 1
warmup 60
rconpassword wtf
// optimization
h_exec optimiza
minimize


Чем он лучше/хуже обычного NFK сервера?
Я как понял, этот trx сервер является ботом bot.dll с какими-то командами и сомнительной полезности функционалом. Который имеет свой ехешник и запускает SERVER.dat (он же переименованный nfk.exe) с параметрами +gowindow +nosound +nfkplanet +game server +exec server +dontsavecfg software

Так вот, кто в теме, поделитесь пошаговой инструкцией по настройке стабильного дедика. Каким образом нужно его настроить, чтобы он нормально функционировал и не лагал?
И хотелось бы актуальные конфиги для всех режимов и наиболее популярные карты, с которыми люди играют.
