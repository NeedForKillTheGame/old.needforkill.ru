Уважаемый господин Патрон любезно согласился корректно перевести на английский следующий текст,
который будет добавлен на американский игровой сайт для рекламы...

Слова можно переставлять как угодно менять, однако смысл джолжен остаться темже...

Описание некоторых изменений версии:
* Полностью изменена графика оружия/интемов/бриков
* Добавлена поддержка спрайтов
* Исправлен глюк на 6 игроков, теперь игра сободно поддерживает
до 8 игроков
* Пофикшено множество багов
* Добавлено несколько новых карт / изменены старые
* Добавлен чистый NFK.exe для любителей обычной NFK
* Добавлено несколько новых моделей
* Добавлена новая нфк планета
+ И множество других менее важных изменений...

Перед началом игры мы советуем отредактировать под себя
файл BOTS\autoexec.cfg
И прочесть файлик DOCS\Info XENG.txt

Примечание:
TRX, SKY & AST MAPS - I's Tribes-X maps and to play on in you must START ONLY Tribes-X.exe ! (NFK does not support

this maps)
NFK & XMP MAPS - To play on this maps you can use NFK.exe or Tribes.exe (TRIBES - support all maps)

Good Luck !

Some new version changes:
* Completely changed graphics of weapons/items/bricks
* Sprite support added
* 6+ player bug fixed. Game supports up to 8 players game.
* A lot of smaller bugs are fixed.
* Some new maps are added. Some old ones are renovated.
* A "clean" NFK.exe is added for those who prefer default NFK engine.
* Some new models are added.
* Now NFK Planet is added
...And many other lesser fixes which are not listed here.

Before starting the game, we would recommend you to edit BOTS/autoexec.cfg
for your own easement of playing in future.
And to read DOCS/Info XENG.txt.

Note:
TRX, SKY, AST MAPS - These are maps designed only for Tribes-X 2.0. In order to play
them, please run only the appropriate .exe, NFK DOES NOT SUPPORT THEM.

NFK, XMP MAPS - To play these ones, you can use both - NFK and Tribes.
