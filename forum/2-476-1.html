Разработка статистики продолжается, реализована уже основная часть, работает вроде стабильно.
Страница статистики: http://nfk.pro2d.ru/stats/index.php
Чтобы участвовать в статистике нужно сыграть на одном из дедиков на nfk planet.
Свои предложения по статистике можно писать тут.

Основные возможности статистики:
+ Просмотр всех прошедших матчей.
+ Загрузка демки матча.
+ Можно смотреть кто сейчас находится на сервере.
+ Подсчет очков в дуэлях по системе iCCup.
+ Список результатов последних дуэлей в матче.
+ В дуэлях введено ограничение на количество побед над одним игроком в сутки: максимум 4 победы.

Доступные комманды на дедиках:
+ !maps (или !m) - выводит список карт имеющихся на сервере, например !maps ctf (tdm, dom, dm, duel)
+ !help (или сокращенно !?) - выводит список доступных команд
+ !vote (или !v) - выводит список возможных голосований на дедике. (например, для кика - !vote kick)
+ !score (или !s) - показывает дуэльные очки игроков.
+ !version - версия статистики
Все команды писать в чат.

Теперь на дедике Rip #1 All можно голосовать за кик игрока
Чтобы начать голосование нужно в чат отправить сообщение: !kick PlayerID
где PlayerID - номер игрока на сервере.
Если написать просто !kick, то выведется список игроков с номерами ID, по которым можно кикать
Чтобы голосовать за или против, отправляем сообщение !yes или !no

Начать голосование можно если только на сервере не меньше 4 игроков.
