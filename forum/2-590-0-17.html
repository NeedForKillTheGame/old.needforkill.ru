Программы для игроков

- NFK Lobby (лучший чат + сканер планеты + выбор предпочтений режимов игры)
Автор: nurupo
https://github.com/nurupo/nfk-lobby
http://needforkill.ru/forum/25-595-1

- Need For Talk (удобный сканер планеты вкупе с IRC чатом)
Автор: rejjin
https://bitbucket.org/rejjin/nft/
http://needforkill.ru/load/need_for_talk/12-1-0-599

- Planet Scanner (устаревшая программа для поиска серверов на NFK Planet)
Автор: booblik
https://bitbucket.org/coolant/nfk-planet-scanner
http://needforkill.ru/load/planet_scanner_1_6/47-1-0-587

- NFK Helper Library (библиотека .NET для извлечения статистики из демок, изменения/создания карт)
Автор: HarpyWar
https://github.com/HarpyWar/nfklib

Программы для сервера
