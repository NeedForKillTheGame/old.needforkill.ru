Игровое меню

Его можно максимально приблизить к оригиналу. К примеру разделить по группам настройки

SETUP
= Controls
== Look
== Move
== Shoot
== Misc

и другие

Также можно сделать более удобным переключение между настройками игрока 1/2.

При выборе Multiplayer (как уже говорилось), надо сразу перекидывать на список лан игр и кнопками переключения Create, Specify. Первое время в R2 по сети будет удобнее играть через Hamachi.
