Уже 4 тестера получат первые аккаунты в nfkLive

Из того, что сделано за сегодня:

можно будет авторизоваться прямо из игры командой

Code
nfklive_auth <login> <password>

Те, кто не особо боятся, смогут разместить эту строчку в свои autoexec конфиги

На сайте nfkLive появилась кнопка Create Server, видимая авторизованными на страничке пользователями. Она будет запускать игру прямо в меню создания сервера! В базу теперь попадает намного больше статистики чем до этого.

Тестирование уже скоро!

Теперь будет отображаться наиболее используемое вами оружие и самый играемый gametype (dm,tdm,ctf,...)
