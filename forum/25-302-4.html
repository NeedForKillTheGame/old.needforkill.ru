 Добавил нофую фитчу
back_brick
Теперь ненадо мутить кучу бак бриков и писать здоровенные конфиги
Теперь можно просто нарисовать на карте чё нам там необходимо и указать что этот брик будет фоновым,

Допустим скрипт всех лестниц на карте может выглядеть так:

back_brick ID
active_brick ID
============

Скрипт всех джамперов так:

back_brick ID
jumper_brick ID
============

Скрипт любых бриков которые мы хотим использовать как бак

back_brick ID

+ Ещё одна фитча, пригодная для синглов
короче если ты в красной команде и наступиш на
так называемый Point_brick то в следующий раз ты респаунишся именно с этой точки и не тока ты а всеь красный тим эта фитча удаляет все красные респы и ставит ровно 1 на том месте где через этот Point_brick ктото из красных тимеров...

Короче используется Легко и доступно...
Допустим в палитре выбираем какойто 1 брик который будет служить нам
этим Пойнтом разставляем эти брики на карте... Через обычный радиант
далее в конфиге добавляем ровно 1 строку
point_brick ID нашего брика
всё )) Теперь если играть CTF
при прохождении через этот бак брик мы автоматически будем распаунится отсюда ))
